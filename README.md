# AngularAdminBlog

This project was generated with [Angular CLI](https://github.com/angular/angular-cli) version 13.3.7.

1. Розробити 2 компоненти: blog та adminBlog

2. Компонента blog це відображення самих блогів, так як бачить їх рядовий користувач

3. Компонента adminBlog, це адмін компонента через яку можна добавляти, редагувати та видаляти блоги

4. Всі дані мають зберігатися в json-server

5. Http-методи мають знаходитися на blog.service.ts

6. Пост не має добавлятися з пустими полями
